Express Entry



Express Entry is an online system that Federal Government of Canada uses to manage applications for permanent residence from skilled workers. Job categories under NOC Level O, A, B qualify for Express Entry. It is a fast track program to get Permanent Residency (PR) in Canada.

There are three categories in Express Entry, namely:
  1. Federal Skilled Worker
  2. Federal Skilled Trades
  3. Canadian Experience Class
Candidates would submit a profile to express entry pool using the Comprehensive Ranking System (CRS). The top ranked candidates are invited to then apply for Permanent Residence.
